## 1. Introduction
A modern Progressive Web Application (PWA) for community event management, built with vanilla web technologies.

## 2. Problem Definition
Communities lack a lightweight, free, privacy-respecting platform for event management.

## 3. Objectives
1. User-friendly event management web app
2. User authentication (registration, login, sessions)
3. Event CRUD with permission control
4. RSVP and attendee tracking
5. Search and filtering
6. PWA offline support
7. Modern glassmorphic UI

## 4. Modules
1. **Authentication** — Registration, Login, Sessions, Password hashing
2. **Event Management** — Create, Edit, Delete events with permissions
3. **RSVP System** — Join/leave events, attendee tracking
4. **Search & Filter** — Text search, category/date filters
5. **PWA** — Service Worker, Cache API, Manifest
