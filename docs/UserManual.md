# User Manual
## Community Event Planner

---

## 1. Getting Started
Open `index.html` in browser directly, or use a local server (`python -m http.server 8000`) for full PWA features.

## 2. Creating an Account
1. Click **"Sign Up"** on login page
2. Fill: Full Name, Email, Password, Confirm Password
3. Click **"Create Account"**
4. You'll be redirected to login

## 3. Logging In
1. Enter Email and Password
2. (Optional) Check "Remember me"
3. Click **"Sign In"**

## 4. Dashboard
- **Header**: App title, your profile, Logout, + Create Event
- **Search Bar**: Type to search events in real-time
- **Filters**: Category dropdown, Date dropdown
- **Events Grid**: All events displayed as cards

## 5. Creating an Event
1. Click **"+ Create Event"**
2. Fill: Title, Category, Date, Time, Location, Description
3. Click **"Save Event"**

## 6. Viewing Event Details
1. Click any event card
2. See full details, attendees list, and action buttons
